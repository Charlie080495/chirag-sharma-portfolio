# Chirag Sharma — Portfolio

Personal portfolio website for **Chirag Sharma**, Senior Business Analyst with 8+ years across Healthcare, Pharma, LMS, and CRM SaaS.

🔗 **Live site:** https://charlie080495.github.io/chirag-sharma-portfolio/

## Highlights

- Single-page portfolio with sticky navigation, smooth scroll, and scroll-triggered reveal animations
- Dark theme by default with a one-click light theme toggle (persists in `localStorage`)
- Eight sections: Hero, About, Skills & Tools, Experience timeline, Case Studies, Certifications, Resume Download, Contact
- Printable web resume at `/resume.html` (prints to PDF cleanly from any browser)
- Mobile-responsive, accessibility-conscious, SEO-friendly (semantic HTML, Open Graph, Schema.org `Person`)

## Tech Stack

Vanilla **HTML5**, modern **CSS** (custom properties, grid, flex), and a small amount of **JavaScript** (IntersectionObserver for animations). No build step. No npm dependencies. Hosts anywhere that serves static files.

## Run Locally

Just open `index.html` in any browser — there's nothing to install.

## Deploy

The site is a fully static bundle, so it works on any static host. The fastest free options:

- **GitHub Pages** — push to GitHub, enable Pages in repo Settings → Pages → Source: `main` / root
- **Netlify** — drag the folder onto [app.netlify.com/drop](https://app.netlify.com/drop)
- **Vercel** — import the repo at [vercel.com/new](https://vercel.com/new)
- **Cloudflare Pages** — connect the repo at [pages.cloudflare.com](https://pages.cloudflare.com)
